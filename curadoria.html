Mini Curadoria de Jogos

O que é isso?

Aqui está uma pequena lista de jogos relacionados á áreas de estudos



Programação

Shezen I/O

Jogo de simulação de programação de dispositivos eletronicos.



while True: learn()

Nesse jogo, você trabalha como um programador que descobre acidentalmente que o seu gato é muito bom em programar, mas não tão bom em falar língua humana. Agora esse programador (você mesmo!) precisa aprender tudo sobre aprendizado de máquina e usar programação visual para criar um sistema de reconhecimento de linguagem de gato para humano.



Comet 64

Comet 64 é um jogo de quebra-cabeça de programação. O ano é 1984. Tudo que você tem é um cometa 64 computador doméstico e o manual do operador que o acompanha.



MHRD

MHRD é um jogo de design de hardware, no qual você projeta vários circuitos de hardware em uma linguagem de descrição de hardware. Os circuitos de hardware que você projeta ficam mais complexos à medida que avança, até que você crie um projeto de CPU totalmente funcional.



Gerenciamento

Prison Architect

Arquitetura e gerenciamento de prisões? Foucault corre aqui!



Tropico 4

Sim, eu sei, problematicamente uma visão da política e cultura caribenha, porém, divertido demais.



Relaxantes

Townscaper

Só construir a cidadezinha, sem desafios, só visual bonito.
